Мета роботи:
Розв’язати нелінійні рівняння та здійснити чисельне інтегрування функцій наближеними методами, використовуючи мови функціонального програмування та lambda, let та set! форми.
Постановка завдання:
1. Написати процедури, що знаходять корені нелінійних рівнянь, використовуючи форми lambda, let, set!
2. Написати процедури, що обчислюють інтеграл функції за формулами прямокутників, трапецій, Сімпсона (парабол)
Середовище Dr.Racket:
Dr.Racket підтримує ряд діалектів та стандартів, в тому числі діалект функціональної мови програмування Scheme, яка якраз вивчалася та використовувалася в ході виконання даної роботи.

; функція f(x) для першої задачі
(define (f x)
; x^2 - e^(-x)
(- (* x x) (exp (- x)))
)
; функція для введення значень
(define (enter-value)
(string->number (read-line))
)
; функція для методу перебору
(define (selec a b i n eps)
(define x (+ a (* i (/ (- b a) (+ n 1)))))
(if (> i n)
(display "Не вдалось знайти рішення")
(if (= (* eps (round (/ (f x) eps))) 0)
(* x 1.0)
(selec a b (+ i 1) n eps)
)
)
)
; функція для методу хорд
(define (chord x0 x1 eps)
(define x (- x1 (* (f x1) (/ (- x1 x0) (- (f x1) (f x0))))))
(if (= (* eps (round (/ (f x) eps))) 0)
x
(begin
(set! x0 x1)
(set! x1 x)
(chord x0 x1 eps)
)
)
)

(newline)

(newline)

(newline)
)
; функція f(x) для другої задачі
(define (y x)
(/ 1 (+ 1 (* x x)))
)
; функція розрахунку значення хі
(define (xi a b n i)
(+ a (* i (/ (- b a) n)))
)
; функція для формули прямокутників
(define (rectangle a b n i)
(define x1 (xi a b n i))
(define x2 (xi a b n (- i 1)))
(if (> i 1)
(+ (* (/ (- b a) n) (y (/ (+ x1 x2) 2))) (rectangle a b n (- i 1)))
(* (/ (- b a) n) (y (/ (+ x1 x2) 2)))
)
)
; функція для формули трапецій
(define (trapezoid a b n i)
(define x (xi a b n i))
(if (> i 0)
(+ (* (/ (- b a) n) (y x)) (trapezoid a b n (- i 1)))
(+ (* (/ (- b a) (* 1 n)) (/ (y x) 2)) (* (/ (- b a) (* 1 n)) (/ (y (xi a b n n)) 2)))
)
)

Результат роботи програми:
Аналіз достовірності результатів:
Вводимо рівняння з першого завдання в онлайн-калькулятор. Бачимо, що результати майже співпадають.
У другому завданні результати, знайдені за формулами прямокутників та трапецій порівнюються з результатом з онлайн-калькулятора. Бачимо, що результати майже співпадають, отже програма працює правильно.
Висновки:
У цій роботі було вирішено дві задачі: у першій було знайдено розв'язок заданого рівняння двома методами: методом хорд та методом перебору. Виходячи з результатів, більш точним виявився метод хорд. У другій задачі було обчислено інтеграл на заданому проміжку за формулами прямокутників і трапецій. Більш точним виявився перший метод. Також у першій задачі біла використана функція set! для присвоєння нових значень існуючих змінних.
