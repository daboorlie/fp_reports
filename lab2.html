Мета роботи:
    Опанувати теоретичні основи застосування рекурентних співвідношень для обчислення тригонометричних, експоненціальних, степеневих функцій та розробити програми функціональними мовам програмування для обчислення їх значень
Постановка завдання:
    1. Написати процедури, що обчислюють задану функцію за допомогою рекурентних послідовностей, розвинувши її у ряд Маклорена (або Тейлора).
    2. Параметр функції має змінюватися від заданого в процесі виклику мінімального значення до максимального значення із певним кроком.
    3. Розвинення функції в ряд здійснювати із заданою точністю. Точність розрахунку задавати в діапазоні від 10-2 до 10-6.
    4. Для розвинення функції у ряд Маклорена (абоТейлора) створити власну функцію, яка розраховує суму ряду за рекурентним співвідношенням.
    5. Значення функції tg(x) обчислювати через функції sin(x) та cos(x).
    6. Визначити похибку обчислення наближених значень функції як різницю абсолютних значень наближеного обчислення та стандартного значення функції.
    7. Стандартне значення функції обчислювати за допомогою бібліотечних математичних функцій.
Середовище Dr.Racket:
    Dr.Racket підтримує ряд діалектів та стандартів, в тому числі діалект функціональної мови програмування Scheme, яка якраз вивчалася та використовувалася в ході виконання даної роботи.

Код програми:
#lang racket
(define (task1 n) ; ініціалізація функції для задачі 1
(count -2 n)
)
(define (count x n) ; функція для виведення обчислених значень
(cond ((<= x 2) (begin
(display "x = ")
(display x)
(newline)
(display(sol1 x n)) ; виводимо рішення через ряд Тейлора
(newline)
(display(sol2 x)) ; виводимо рішення через вбудовану функцію
(newline)
(newline)
(count (+ x 0.5) n)
)
))
)
(define (sol1 x n) ; функція для знаходження рішення через ряд Тейлора
(cond ((and (>= x 1) (<= x 2)) (sqrtc (- 15 (* x x)) n))
((and (>= x -1) (< x 1)) (/ 1 (sqrtc (+ x (* x x)) n)))
)
)
(define (sol2 x) ; функція для знаходження рішення через вбудовану функцію
(cond ((and (>= x 1) (<= x 2)) (sqrt (- 15 (* x x))))
((and (>= x -1) (< x 1)) (/ 1 (sqrt (+ x (* x x)))))
)
)
(define (sqrtc x n) ; функція для розрахунку квадратного кореня через ряд Тейлора
(if (= n 0)
1
(* 0.5 (+ (sqrtc x (- n 1)) (/ x (sqrtc x (- n 1)))))
)
)
(define (task2 a) ; ініціалізація функції для задачі 2
(sol a 1)
)
(define (sol a n) ; функція розрахунку рішення виразу
(if (< n 256)
(/ n (+ (* a a) (sol a (* n 2)))) ; n / (a^2 + sol(a, n*2))
(* (/ n (* a a)) 1.0) ; n / a^2
)
)
Результат роботи програми:
    В результаті маємо рішення двох завдань. В першому завданні виводиться рішення на заданому проміжку двома способами: через ряд Тейлора та через вбудовану функцію. В другому завданні обчислюється рішення скінченого рекурентного виразу.
Висновки:
    В результаті роботи було вирішено дві заданні програмні задачі за допомогою мови Racket. У першій задачі програма на деяких значеннях видавала значення void або комплексне число через те, що в розрахунках попадалися значення корня числа менше нуля або через ділення на 0. У другій задачі програма обчислює значення скінченного рекурентного виразу.
